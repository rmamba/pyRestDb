Python RestDB

Created for RaspberryPI to store NMEA GPS parsed locations in memory instead of writing them constantly to SD card. You can find pyNMEA-RaspberryPi NMEA parser project on github.com

Stored data:

Usage:
All links return data in JSON format (add (?&)pjson to return pretty JSON)
Store NUMBER value: http://domain:port/set/{variable}/{number}
Store STRING value: http://domain:port/set/{variable}/"{string}"
Store complex values with POST (can be JSON data): http://domain:port/post/{variable}
Get value: http://domain:port/get/{variable}
Store hidden NUMBER varlue: http://domain:port/set/{variable}/{value}?secret={HASH}
Store hidden STRING varlue: http://domain:port/set/{variable}/{value}?secret={HASH}
Get hidden value: http://domain:port/get/{variable}?secret={HASH}

Admin Usage:
All links return data in JSON format
Show hidden data: http://domain:port/admin/{password}
Delete value: http://domain:port/admin/delete/{password}/{variable}
Delete hidden value: http://domain:port/admin/delete/{password}/{variable}?secret={HASH}
Purge all values: http://domain:port/admin/purge/{password}
Purge only hidden values: http://domain:port/admin/purge/{password}?secret={HASH}
