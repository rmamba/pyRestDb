Python RestDB

Created for RaspberryPI to store NMEA GPS parsed locations in memory instead of writing them constantly to SD card.

Stored data

Usage:

All links return data in JSON format

Store NUMBER value: /set/{variable}/{number}

Store STRING value: /set/{variable}/{string}

Store complex values with POST (can be JSON data): /post/{variable}

Get value: /get/{variable}

Store hidden NUMBER varlue: /set/{variable}/{value}?secret={HASH}

Store hidden STRING varlue: /set/{variable}/{value}?secret={HASH}

Get hidden value: /get/{variable}?secret={HASH}

Admin Usage:

All links return data in JSON format

Show hidden data: /admin/{password}

Delete value: /admin/delete/{password}/{variable}

Delete hidden value: /admin/delete/{password}/{variable}?secret={HASH}

Purge all values: /admin/purge/{password}

Purge only hidden values: /admin/purge/{password}?secret={HASH}
